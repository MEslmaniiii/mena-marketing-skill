## Structure

```
progress/
├── README.md              ← This file
├── tracker.md             ← Update after every completed task
└── your-name/             ← Create a folder with your name
    ├── copywriting/
    ├── seo/
    ├── ads/
    ├── analytics/
    └── growth/
```

---

## File Naming Convention

`[track]-[task-number]-[your-initials]-[date].md`

Example: `copywriting-01-MA-2025-04.md`
