# Task 03: A/B Test Plan — خطة اختبار A/B
**Level:** BUILDER | **Language:** English | **Time:** 45–60 min

---

## The Brief

**Brand:** Mouj — UAE luxury beachwear e-commerce brand
**Goal:** Increase purchase conversion rate on Meta ads
**Current ROAS:** 2.4x (target: 4x)
**Monthly budget:** AED 45,000

---

## Your Task

Design a structured A/B test plan for the next 30 days on Meta.

### Each Test Must Include:

1. **Hypothesis** — "We believe [changing X] will [result in Y] because [reason]"
2. **Variable being tested** — Exactly one variable per test
3. **Control** — What you're testing against
4. **Variant** — What you're testing
5. **Audience** — Who sees this test
6. **Budget allocation** — How much to each side
7. **Duration** — How long to run before reading results
8. **Success metric** — Primary KPI to determine winner
9. **Minimum threshold** — What result makes this statistically meaningful?

---

## Design 3 Sequential Tests

**Test 1: Creative Format** (Week 1–2)
Test whether video outperforms static for this audience.

**Test 2: Audience** (Week 2–3)
Test broad audience vs. interest-based targeting.

**Test 3: Offer/CTA** (Week 3–4)
Test two different offers or CTAs.

---

## Constraints

- Only one variable per test — do not change multiple things at once
- Each test needs a clear winner criterion before you start
- Tests must be sequential, not simultaneous (budget is limited)
- Include what happens if results are inconclusive

---

Ask Claude: *"Review my A/B test plan for a UAE e-commerce Meta campaign. Are my hypotheses testable? Is my methodology sound?"*
