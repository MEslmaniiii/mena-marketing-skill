## The MENA Copywriter's Mindset

Most Arabic copy fails because it's either:
1. **Translated from English** — sounds stiff, unnatural
2. **Too formal** — Fusha in a context that calls for dialect
3. **Vague** — "أفضل جودة" says nothing
4. **Feature-first** — lists what the product does, not what it fixes

Great Arabic copy is **specific, direct, and sounds like the audience talks**.

## The Copywriting Formula for MENA

```
HOOK      → Stop the scroll. State the problem or outcome in 1 line.
PROBLEM   → Name exactly what's broken or frustrating.
SHIFT     → Reframe how they should think about the solution.
PROOF     → Show evidence (numbers, name, story).
OFFER     → State exactly what they get.
CTA       → Tell them the one thing to do next.
```
